# Lokální zdrojové dokumenty

Tato složka může lokálně obsahovat licencované nebo autorsky chráněné podklady, které slouží pouze k vytvoření osnovy kurzu. PDF soubory jsou ignorované Gitem a nesmějí být publikovány ani zahrnuty do produkčního buildu. Do obsahu aplikace se nepřebírá text učebnice, pouze názvy a hierarchie témat.
